# Reflection

This project improved practical understanding of network security, firewall configuration and cybersecurity implementation.

OpenWRT deployment provided experience in configuring virtual networks, routing and firewall management.

Firewall testing demonstrated how blocking HTTP traffic, ICMP requests and changing SSH ports can reduce security risks.

Traffic analysis using tcpdump and Wireshark showed differences between secure protocols such as SSH and insecure protocols like HTTP.

The project also improved skills in:

- Linux commands
- Network troubleshooting
- Virtualisation
- Packet analysis
- GitHub collaboration

Overall, the project highlighted the importance of implementing layered security controls in organisations handling sensitive financial information.
