## Project Overview

This project demonstrates implementation of cybersecurity practices for a small accounting business using OpenWRT and VirtualBox.

The project includes:

- OpenWRT deployment
- Firewall configuration
- SSH hardening
- Password security
- Traffic analysis
- Packet capture
- Risk assessment
- Production network design
- Security controls
